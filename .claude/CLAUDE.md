# AI-CLI - Git 提交规范

## 版本号规则

- 格式：`v{主版本}.{次版本}.{修订版本}`
- 新版本号 = 最近提交版本号 +1
- 获取最新版本号：`git log --oneline -1`

## 提交说明格式

```
{版本号}: {修改内容描述}
```

**示例：**
```
v2.2.4: 修复用户登录会话超时问题
v2.2.3: 添加 CLAUDE.md 配置文件定义 Git 提交规范
v2.2.2: 实现文档多语言支持并优化目录结构
```

## 生成提交说明步骤

1. `git log --oneline -5` 读取最近提交历史
2. 解析最新版本号并递增修订版本
3. 分析变更内容，生成简洁中文描述
4. `git commit -m "vX.Y.Z: 描述"`

## 变更类型前缀

| 类型 | 前缀 | 示例 |
|------|------|------|
| Bug 修复 | 修复... | `v2.2.1: 修复空文件夹无法新增项目的问题` |
| 新功能 | 实现.../增加... | `v2.2.0: 实现树状结构管理` |
| 优化 | 优化.../改进... | `v2.2.1: 优化新增项目界面` |
| 文档 | 更新.../添加... | `v2.2.2: 实现文档多语言支持` |
